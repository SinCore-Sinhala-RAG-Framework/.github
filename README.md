  <h1>SinCore</h1>

  <section>
    <h2>Overview</h2>
    <p>
      <strong>SinCore</strong> is a native-script Retrieval-Augmented Generation (RAG) framework for the Sinhala language. It addresses current limitations of transliteration-dependent and multilingual approaches, providing a morphology-aware, script-sensitive RAG system supported by domain-diverse Sinhala corpora and lightweight fact-aware retrieval.
    </p>
  </section>

  <section>
    <h2>Motivation</h2>
    <p>
      While RAG frameworks have improved factual reliability in languages like Arabic and Swahili, Sinhala remains underserved due to:
    </p>
    <ul>
      <li>Fragmented corpora</li>
      <li>Lack of morphology-aware embeddings</li>
      <li>Absence of culturally adapted evaluation metrics</li>
    </ul>
    <p>SinCore aims to fill this gap and provide a robust solution for the 16+ million Sinhala speakers.</p>
  </section>

  <section>
    <h2>Methodology</h2>
    <ol>
      <li>Corpus development with domain-diverse Sinhala texts</li>
      <li>Morphology-aware tokenization</li>
      <li>Embedding optimization for script-sensitive representations</li>
      <li>Integration of retrieval mechanisms with LLM generation</li>
    </ol>
  </section>

  <section>
    <h2>Expected Outcomes</h2>
    <ul>
      <li>A benchmarked Sinhala RAG framework</li>
      <li>Novel script-aware evaluation metrics</li>
      <li>Scalable design principles applicable to other low-resource South Asian languages</li>
    </ul>
  </section>
